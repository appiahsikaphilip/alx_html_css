## Figma Design Notes

- Font families: **Source Sans Pro** and **Spin Cycle OT**  
- Floating-point values in the design may be **rounded**  
- Desktop max width: **1000px**  
- Mobile version triggers at **480px or less**  
- Interactions:
  - Links hover/active: `#FF6565`
  - Button hover/active: `opacity: 0.9`
